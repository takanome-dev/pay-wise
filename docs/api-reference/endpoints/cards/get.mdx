---
title: 'Get all user cards'
api: 'GET /api/cards'
description: 'This endpoint gets all the cards for a user.'
---

### Response

<ResponseField name="cards" type="array">

The user's cards.

<Expandable title="Toggle object">

<ResponseField name="card)number" type="string">
  This is the card number of the card. This is the number that is displayed on
  the card.
</ResponseField>

<ResponseField name="cvv" type="string">
  This is the CVV of the card. This is the number that is displayed on the card.
</ResponseField>

<ResponseField name="expiry" type="string">
  This is the expiry of the card. This is the number that is displayed on the
  card.
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.mintlify.com/api/user' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "user_group_id": "example_1",
    "name": "Example 1",
    "mapping": {"40": "213", "134": "386"},
    "properties": {"filterValue": "value"}
}'
```

</RequestExample>
